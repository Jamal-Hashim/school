# 🎓 Nomani School Website

A modern and responsive **School Website** built using **HTML, CSS, and JavaScript**.  
This project represents a complete educational website that provides information about the school, admissions, classes, teachers, and contact details.

The website is designed with a **clean user interface, smooth animations, and responsive layout** to ensure a great experience on both desktop and mobile devices.

---

## 🌟 Features

- 🏫 Beautiful **Home Page with Hero Slider**
- 📖 **About Us Page** with school information
- 📝 **Admission Page** for new student enrollment
- 📚 **Classes Page** showing available classes
- 📞 **Contact Page** with school contact details
- 👨‍🏫 **Teachers Section**
- 🎯 **Mission & Vision Section**
- 📊 **Animated Counters**
- 💬 **Testimonials Section**
- 📱 **Responsive Design for Mobile and Desktop**
- 🎨 Smooth **CSS Animations and Hover Effects**
- 🍔 Mobile **Navigation Menu**

---

## 🛠️ Technologies Used

- **HTML5** – Website structure
- **CSS3** – Styling and layout
- **JavaScript** – Interactivity and slider functionality
- **Font Awesome** – Icons
- **Responsive Web Design**
